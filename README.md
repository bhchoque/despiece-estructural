# Despiece Estructural

Herramienta para generar despieces de vigas estructurales 
desde Revit → Dynamo → JSON → DXF.

## Flujo
1. Exportar vigas desde Revit con el script Dynamo
2. Abrir `despiece.ipynb` en Jupyter
3. Cargar el JSON y generar el DXF

## Requisitos
pip install notebook ezdxf matplotlib ipywidgets
